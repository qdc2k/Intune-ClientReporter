## Features

- Last logged-on user and login history with timestamps
- Enter customer tenants via Domain / Tenant-ID (remains saved and can be selected from a dropdown later)
- All data is cached under %appdata% and does not need to be requested again every time
- Smart and fast search function to search for partial strings (parts of usernames, serial numbers, device names, etc.)
- MAC addresses (Wi-Fi and Ethernet) read out directly
- Export to CSV function
